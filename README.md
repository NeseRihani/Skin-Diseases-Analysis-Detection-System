# 🩺 Skin Diseases Analysis & Detection System

This project is an end-to-end system that analyzes skin imperfections using AI-based object detection (**YOLO**) algorithms and visualizes the results through a web-based interface.

---

## 🚀 Project Overview
The system detects potential skin diseases or lesions from user-uploaded images. Detected data is securely stored in a database and reported through a user-friendly web interface.

## 🏗️ Technical Architecture (N-Tier)
The project is developed using **N-Tier (Layered) Architecture** to ensure sustainability and scalability:

* **Entity Layer:** Object models representing database tables.
* **Data Access Layer (DAL):** The layer where database (SQL) operations (CRUD) are managed.
* **Business Layer:** The layer where business logic and validations are executed.
* **Presentation Layer (Web):** The user interface where users interact with the system.

## 🛠️ Tech Stack
* **Artificial Intelligence:** YOLO (You Only Look Once) - Object Detection Model.
* **Backend:** .NET / C# (N-Tier Architecture).
* **Frontend:** HTML5, CSS3, JavaScript.
* **Database:** SQL Server.
